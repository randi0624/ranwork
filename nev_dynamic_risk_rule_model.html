1. 建模原则

明确排除：vehicle_and_battery_score 中的 battery_overall_score、vehicle_overall_score、driving_behavior_score、motor_score、charge_score、chassis_score 等历史自定义评分，不作为模型输入。
原则说明
只用事实字段使用次数、时长、里程、速度、SOC、电压、温度、报警等级、碰撞前后帧等客观数据。
先规则后校准先由风控专家定义规则和阈值，再用真实理赔结果校准权重。
强解释性每个风险标签都能回溯到具体表、字段、时间和证据值。
分层输出输出综合分、维度分、风险标签、证据链和保司动作建议。

2. 综合风险指数

分数等级含义建议动作
0-30低风险动态行为稳定，无严重故障或事故链路。正常承保/续保。
31-60中风险存在单项风险或轻度异常。展示关注项，必要时费率微调。
61-80高风险多项异常叠加，或单项严重风险。人工核保，限制责任或调整免赔。
81-100极高风险疑似真实事故、严重故障、电池安全隐患或道德风险。强制复核，必要时拒保/理赔调查。
risk_score = driving_behavior_score * 0.18 + collision_accident_score * 0.18 + battery_safety_score * 0.16 + fault_alarm_score * 0.14 + charging_behavior_score * 0.10 + motor_powertrain_score * 0.08 + exposure_scenario_score * 0.08 + abnormal_moral_hazard_score * 0.08

强规则：若疑似真实事故、三级故障持续、制动/高压互锁严重报警、电池温差/压差严重异常、碰撞前异常加速且事故后停车熄火，则综合分直接拉升到高风险或极高风险。

3. 八大风险维度

驾驶行为 18%

超速、疲劳、分心、危险驾驶、急加速、急减速、急转弯、夜间、低 SOC 行驶。

疑似事故/碰撞 18%

碰撞预警、碰撞前异常加速、碰撞后停车熄火、开门、传感器故障、前后 100 秒帧。

电池安全 16%

过充、过放、高温暴露、绝缘异常、单体电压极差、温度极差、带故障行驶。

故障报警 14%

国标报警等级、报警类型、持续时长、报警帧数、故障期间速度/SOC/电池异常。

充电行为 10%

高频充电、高频快充、低电量充电、夜间充电、充电功率波动。

电机动力 8%

电机高温、控制器高温、电机故障数、输出功率峰值与功率冲击。

暴露场景 8%

总里程、总时长、夜间、高峰、风险天气、跨区域、长距离、长时间行驶。

异常/道德风险 8%

事故前反复经过地点、事故后行为链路异常、报案与动态证据冲突、传感器故障异常。

4. 具体算法规则

4.1 驾驶行为风险

规则事实字段评分口径
超速风险is_over_speed_driving、over_speed_nums、drun_max_speed命中月度超速 +15；日超速次数按百公里归一，最高 +25；最大车速异常 +10。
疲劳与长时间行驶is_high_freq_fatigue_driving、fatigue_driving_nums、long_time_run_nums疲劳驾驶命中 +20；长时间行驶频繁 +10~20；夜间叠加 +5。
三急行为rapid_deceleration_nums、rapid_acceleration_nums、sharp_turn_nums按百公里次数归一：急减速权重 1.2，急加速 1.0，急转弯 1.1。
分心/危险驾驶distracted_driving_nums、dangerous_driving_nums高频分心 +20；高频危险驾驶 +30；两者同时命中额外 +10。
driving_behavior_score = min(100, 15 * is_over_speed_driving + 20 * is_high_freq_fatigue_driving + 20 * is_high_freq_distracted_driving + 30 * is_high_freq_dangerous_driving + per100km(rapid_deceleration_nums) * 1.2 + per100km(rapid_acceleration_nums) * 1.0 + per100km(sharp_turn_nums) * 1.1 + 8 * is_deep_night_driving + 8 * is_high_freq_low_soc_driving)

4.2 疑似事故与碰撞风险

规则事实字段评分口径
疑似真实事故cw_after_5min_is_fact_accident、cw_after_5min_zero_speed_time、cw_after_5min_stop_time疑似真实事故 +60；5 分钟内停车/熄火 +20。
碰撞前异常驾驶crash_before_is_unusual_add_speed、cw_before_2min_is_unsafe_driving、cw_before_10min_driving_warn_nums异常加速 +35；2 分钟内不安全驾驶 +25；10 分钟预警每次 +4，封顶 +20。
碰撞后行为异常cw_after_6h_sensor_fault_time_list、cw_after_first_stop_time_length、cw_trip_after_6h_door_open_off_nums传感器故障 +20；停留异常 +10~20；车门开关异常 +5~15。
碰撞帧证据frame_data_speed、frame_data_brake_effort、frame_data_accelerator_pedal_position用于比对事故描述与责任判断；冲突时进入道德风险强规则。

4.3 电池安全风险

规则事实字段评分口径
过充过放over_charge_nums、over_discharge_nums、over_charge_discharge_nums每次 +5，月度封顶 +35；过充过放同时出现额外 +10。
高温暴露high_temperature_nums每次 +4，封顶 +30；高温叠加快充额外 +10。
电压/温度极差alarm_max_cell_voltage_diff、alarm_max_cell_temp_diff按阈值分段：轻度 +15，中度 +35，严重 +70。
绝缘/高压风险hvil_alarm_nums、国标绝缘相关 alarm_type高压互锁每次 +15；绝缘相关高等级告警直接高风险。

4.4 故障报警风险

规则事实字段评分口径
报警等级alarm_level、alarm_max_level1 级 +15，2 级 +35，3 级 +70。
带故障行驶with_alarm_run_nums、drun_with_alarm_run_mileage次数 + 里程双因子；带故障行驶超过 20km 额外 +20。
制动系统报警brake_system_alarm_nums每次 +20，封顶 +80。
报警持续性start_time、end_time、alarm_contain_frames持续时间越长、帧数越多，风险越高。

4.5 充电、电机和暴露规则

维度事实字段评分口径
充电习惯is_high_freq_charging、is_high_freq_fast_charging、is_high_freq_low_soc_charging高频充电 +10；高频快充 +20；低 SOC 充电 +20；三者叠加进入中高风险。
电机温度motor_temp_too_high、motor_temp_max、motor_controller_temp_too_high温度过高次数每次 +8；最高温超阈值 +15~35。
电机故障motor_fault_nums_max、motor_fault_nums_avg故障数大于 0 即 +20；高频或持续故障 +40。
暴露场景total_run_mileage、total_run_time_length、night_run_nums、risk_weather_run_nums高里程/高时长/夜间/风险天气/跨区域分段加分。

5. 保司证据链展示

风险标签证据字段展示文案动作建议
高频危险驾驶dangerous_driving_nums、is_high_freq_dangerous_driving近 30 天危险驾驶行为高于阈值。核保关注驾驶风险。
疑似真实事故cw_after_5min_is_fact_accident、cw_after_5min_zero_speed_time碰撞预警后车辆停车/熄火，疑似真实事故。进入理赔调查。
电池安全异常alarm_max_cell_voltage_diff、alarm_max_cell_temp_diff、hvil_alarm_nums电池极差或高压系统存在异常。要求检测/维修证明。
带故障行驶with_alarm_run_nums、drun_with_alarm_run_mileage车辆故障状态下仍持续行驶。提高风险等级。
高频快充低电量is_high_freq_fast_charging、is_high_freq_low_soc_charging充电习惯可能加速电池衰减。纳入电池风险。
